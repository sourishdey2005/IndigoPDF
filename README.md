# IndigoPDF

IndigoPDF is a modern, privacy-first PDF utility suite built with Next.js. It allows users to perform a wide range of PDF operations completely client-side, ensuring that sensitive documents never leave the user's device.

## Key Features
- **Organize**: Merge, Split, and Organize PDF pages.
- **Optimize**: Compress and Repair PDF files.
- **Convert**: Support for Word, Excel, PowerPoint, JPG, and HTML conversions.
- **Edit**: OCR, Rotate, Watermark, and Page Numbering.
- **Security**: Protect, Unlock, Sign, and Redact sensitive information.
- **Advanced**: AI-powered Translation and Scan-to-PDF via device camera.

## Technology Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & Shadcn UI
- **PDF Engine**: `pdf-lib` & `pdf.js`
- **AI Integration**: Genkit (Google AI)
- **Animation**: Framer Motion
