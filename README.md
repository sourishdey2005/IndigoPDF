
# IndigoPDF 🚀

IndigoPDF is a professional-grade, **privacy-first PDF utility suite** built with Next.js 15. It empowers users to perform 25+ essential PDF operations completely client-side. By leveraging the power of the user's browser, IndigoPDF ensures that sensitive documents never leave their device, providing a secure alternative to cloud-based editors.

## 🛡️ Privacy First
Most online PDF tools upload your sensitive documents to their servers for processing. **IndigoPDF is different.** Every operation—merging, splitting, compressing, or signing—happens locally in your browser's memory. 
- **No Uploads**: Your files never touch a server.
- **No Limits**: Process as many files as you want, for free.
- **No Watermarks**: Professional output without the branding bloat.

## ✨ Key Features

### 📂 Organize & Structure
- **Merge PDF**: Combine multiple documents into one in seconds.
- **Split PDF**: Extract specific page ranges or individual pages.
- **Organize & Reorder**: Drag-and-drop page thumbnails to restructure your document.
- **Delete Pages**: Visually select and remove unwanted content.
- **Batch Rename**: Automatically rename large sets of PDFs using custom patterns.

### ⚡ Optimize & Repair
- **Native Compression**: Reduce file size using intelligent re-distillation.
- **Repair PDF**: Fix corrupted PDF structures and recover data.
- **Flatten PDF**: Merge layers and form fields to prevent further editing.
- **Grayscale**: Convert documents to monochrome to save on printing costs.

### 🔄 Conversion Suite
- **To PDF**: Word (.docx), Excel (.xlsx), PowerPoint (.pptx), Images (JPG/PNG), and Webpages (HTML).
- **From PDF**: Convert PDF to Word, Excel, PowerPoint, JPG, Text, or HTML.

### 📝 Advanced Editing
- **Professional OCR**: Extract text from scanned documents using AI-powered recognition.
- **Sign PDF**: Draw or upload your signature and apply it to any page.
- **Watermark**: Stamp documents with custom text overlays.
- **Bates Numbering**: Apply sequential legal-grade numbering.
- **Metadata Editor**: Edit title, author, subject, and keywords.
- **Page Controls**: Precise rotation (0-360°) and margin cropping.

## 🛠️ Technology Stack
- **Framework**: Next.js 15 (App Router)
- **UI Components**: Shadcn UI & Tailwind CSS
- **PDF Core**: `pdf-lib` (Native manipulation) & `pdf.js` (Rendering)
- **AI Integration**: Genkit with Google Gemini (Multimodal OCR)
- **Animation**: Framer Motion
- **Icons**: Lucide React

## 📱 Mobile Support
IndigoPDF is fully responsive and also features a dedicated **Android APK** for processing documents on the go with the same privacy guarantees.
