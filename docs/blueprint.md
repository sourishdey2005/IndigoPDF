# **App Name**: IndigoPDF

## Core Features:

- Merge PDF: Combine multiple PDF files into a single document, offering options for reordering pages client-side.
- Split PDF: Divide a large PDF into multiple smaller files based on user-defined page ranges or intervals, processed entirely in the browser.
- Compress PDF: Reduce PDF file size with adjustable compression levels (Low, Medium, High) and provide before/after comparisons.
- JPG to PDF Conversion: Convert one or more JPG images into a single, optimized PDF document with quality settings and page size selection.
- Protect PDF: Add 128-bit AES encryption and set specific permission controls to a PDF document, handled securely client-side.
- OCR PDF Tool: Utilize an intelligent tool to perform Optical Character Recognition (OCR) on scanned PDFs, transforming them into searchable and editable text documents.
- Unified Tool Workflow: A consistent user interface for selecting tools, intuitively uploading files via drag-drop, configuring specific settings, and downloading processed results.

## Style Guidelines:

- Primary Color: A deep, digital Indigo (#4F46E5) signifying trust, privacy, and modern technology. Used for key interactive elements.
- Secondary Color: A harmonious Violet (#8B5CF6) that complements the primary Indigo, adding visual depth and brand consistency.
- Accent Color: A vibrant Emerald (#10B981) for call-to-action buttons, success indicators, and other elements requiring clear emphasis, providing stark contrast.
- Background Color: A clean and minimal Slate (#F8FAFC) offering a light, spacious canvas that enhances content readability and tool visibility.
- Text Color: A rich, dark Slate (#0F172A) ensuring excellent contrast and legibility across all UI elements and informational text.
- Headings and Body Font: 'Inter' (sans-serif) for its modern, highly legible, and versatile characteristics across various screen sizes.
- File Metadata Font: 'JetBrains Mono' (monospace) for displaying technical information, paths, or code snippets with clear distinction and readability.
- Minimalist, clean line icons from Lucide React, chosen for clarity and their modern aesthetic to align with a professional SaaS interface.
- A responsive, mobile-first SaaS layout featuring rounded corners (8px for inputs, 12px for cards), soft shadows, and glass effect headers for a polished and contemporary feel. Key sections include a prominent hero with an animated illustration, a structured tool grid, and clear 'How It Works' guidance.
- Subtle and fluid micro-interactions using Framer Motion to enhance user experience, along with Lottie animations for engaging and informative processing feedback during tool execution.
